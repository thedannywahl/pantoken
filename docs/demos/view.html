View

.instui-view is InstUI's View. Beyond the neutral box it carries key-value modifiers for its own visual props, so it has the same devex as the other components — no need to reach for utilities for the basics.

Native modifiers

-background-primary -border-radius-large -shadow-resting
-border-width-small -border-color-brand -border-radius-medium
-background-brand

Compose utilities too

Spacing, color, and token utilities still layer onto a View — e.g. instui-bg-accent-blue + instui-fg-on-color + instui-p-md.
